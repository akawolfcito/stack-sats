## Screenshots (cws-01..05.png)

The five 1280x800 store cards are generated by the Playwright spec
`e2e/store-screenshots.spec.ts`. Each card composites a 400x600 popup
capture onto a branded background with the DenVault logo and headline.

To regenerate:

```bash
pnpm exec playwright test e2e/store-screenshots.spec.ts --project=chromium
```

Captions per card (set in the spec):

1. `cws-01-start.png` — Onboarding / start hero
2. `cws-02-home.png` — Wallet home
3. `cws-03-send.png` — Send STX
4. `cws-04-receive.png` — Receive
5. `cws-05-settings.png` — Settings / network
