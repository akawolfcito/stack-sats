DenVault

Secure, non-custodial Stacks wallet for Bitcoin Layer 2

Open-source and built with production-grade security. Independent project by DenLabs. Review the source code before storing significant funds.

AES-256-GCM Encryption PBKDF2 600k Iterations Auto-Lock Protection Non-Custodial Open Source

Multi-Wallet Support

Manage multiple wallets with custom names. Add and remove derived accounts as needed.

SIP-010 Tokens

View all your fungible tokens following the SIP-010 standard with automatic metadata.

dApp Connections

Connect securely to Stacks dApps using @stacks/connect v8 and WBIP protocols.

Encrypted Storage

Your seed phrase is encrypted with AES-256-GCM and protected by a 6-digit PIN.

Network Switching

Seamlessly switch between Mainnet, Testnet, and Devnet environments.

Transaction History

View recent activity and track your transactions across all networks.
