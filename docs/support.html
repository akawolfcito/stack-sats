Frequently Asked Questions

How do I create a wallet?

Click the extension icon, then "Create New Wallet". Write down your 24-word seed phrase securely offline, then set a 6-digit PIN.

How do I import an existing wallet?

Click the extension icon, then "Import Existing Wallet". Enter your 24-word seed phrase and set a new PIN.

I forgot my PIN. What do I do?

You'll need your original 24-word seed phrase. Delete the current wallet and import it again using your seed phrase to set a new PIN.

Why does the wallet auto-lock?

For security, the wallet automatically locks after 5 minutes of inactivity. This prevents unauthorized access if you leave your computer unattended.

What networks are supported?

Mainnet (production), Testnet (for testing), and Devnet (local development). Switch networks via the menu.

How do I connect to a dApp?

Visit a Stacks dApp, click "Connect Wallet", and approve the connection request in the popup. Your addresses will be shared with the dApp.

Is my seed phrase safe?

Your seed phrase is encrypted with AES-256-GCM and protected by your PIN. It's stored only locally and never transmitted to any server.

How do I manage multiple wallets?

Go to Settings and use "Add Wallet" to create or import additional wallets. You can switch between them and give each a custom name.

Troubleshooting

Extension not loading

  1. Go to chrome://extensions
  2. Enable "Developer mode"
  3. Click "Load unpacked" and select the extension folder
  4. Click the reload button if needed

dApp connection not working

  1. Verify you're on localhost or an HTTPS site
  2. Check browser console for errors
  3. Refresh the page and try again
  4. Reload the extension

Transaction failed

  1. Check you have enough STX for amount + fee
  2. Verify the recipient address format
  3. Ensure correct network is selected
  4. Check the blockchain explorer for status
